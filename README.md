### Installation

1. **Clone the repository**
```bash
git clone <repository-url>
cd aminul-haque-portfolio-bnp
```

2. **Install dependencies**
```bash
npm install
```

3. **Setup environment variables**

Create a `.env.local` file in the root (if needed):
```env

```

4. **Start development server**
